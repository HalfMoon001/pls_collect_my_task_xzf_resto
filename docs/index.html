📖 Memo Journal

记忆手帐 Memo Journal

一个带 AI 助理的个人便签手帐应用 ♥

用 Claude Code CLI 作为后端大脑，帮你整理碎碎念、追踪习惯、生成日/周摘要、绘制任务关联图谱。

GitHub

※ Demo 内所有 AI 按钮均为本地静态模拟，不会真的调用 CLI

✨ 它能帮你做什么

📝 随手记录

写下碎碎念、待办、灵感，打标签分类。便签支持编辑、删除、置顶、看板视图。

🤖 AI 整理思路

CC 帮你智能查找、整理标签、生成日/周摘要、绘制任务关联图谱。

🌻 习惯养成

上传任何书籍 ZIP（饮食、运动、观鸟手册…），生成专属习惯卡池，每天抽一张打卡，用可爱胶带贴满日历。

🕯️ 任务盲盒

任务太多，不知从何开始？迷茫时随机抽一条待办，让 AI 帮你决定今天做什么。

⏰ 定时提醒

设置未来日期 + 时间，到点系统通知，重要事项不再错过。

🎨 多款主题

提供 20+ 预设主题色系，支持自定义配色，每个标签可单独设置 emoji 和胶带样式。

🚀 开始使用

🍎 方法一：下载 Mac App

目前仅支持 Mac ARM64（Apple Silicon）。Intel Mac / Windows / Linux 请走方法二。

  1. 先在终端安装并登录 Claude Code CLI：claude → /login
  2. 从 Releases 下载最新 .dmg
  3. 拖到「应用程序」文件夹
  4. 首次打开去「系统设置 → 隐私与安全性」点「仍要打开」
前往 Releases ↗

💻 方法二：Git Clone 运行源码

Mac / Windows / Linux 都支持。需要 Node.js v18+ 和（可选）Claude Code CLI。

git clone https://github.com/AnselKocen/pls_collect_my_task.git
cd pls_collect_my_task
npm install
./start.sh     # Mac / Linux
start.bat      # Windows（或直接双击）

启动后浏览器会自动打开 http://localhost:3013

📖 详细的使用教程（常见问题、数据备份、登录排查等）请前往 GitHub README 查看详细说明。

❓ 小贴士
AI 按钮没反应？ 大概率是 Claude CLI 登录态过期。终端跑一次 claude 重新登录即可。
不想用 AI 也能用吗？ 可以。便签管理、标签、搜索、看板、主题、定时提醒都能用，只是 AI 部分不可用。
数据会丢吗？ 不会。数据在 data/ 或 ~/Library/Application Support/记忆手帐/，升级不会动它。

📖 使用指南
📖

记忆手帐 · 使用指南

✨ 基础功能

• 快速记录：左侧「快速记录」框写下碎碎念，选择标签后点保存

• 便签管理：支持编辑、删除、置顶，点击便签可查看详情

• 标签筛选：顶部标签栏可按标签过滤便签

🤖 智能助手（需要 Claude Code CLI）

• 智能查找：输入你想找的东西，AI 帮你找出相关的任务

• 整理标签：AI 自动分析便签，建议合并或新增标签

• 任务图谱：可视化展示便签之间的关联关系

• 本周总结：每周日 23:59 自动生成过去一周的深度总结与标签趋势

• 今日摘要：每天定时汇总最近两天的手帐内容

🌻 特色玩法

• 习惯养成：上传书籍 ZIP，每天抽取习惯卡打卡

• 任务盲盒：迷茫时随机抽取一条待办，让 AI 帮你做决定

• 定时提醒：设置未来时间点，到点会收到系统通知
