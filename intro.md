# 我 vibe 了一个本地便签工具，让 CC 帮我整理碎碎念、养成任何习惯、甚至学武则天的权谋心法

我 vibe 了一个带 AI 小助理的便签手帐 app——**记忆手帐 Memo Journal**，CC 住在里面帮我整理乱七八糟的碎碎念、追踪习惯、生成摘要、画任务关联图谱。

## 为什么要做这个

我平时就老想着，每次想让 CC 帮忙干点什么，都得单开一个终端窗口，对着黑底白字的 terminal 敲命令……能用是能用，但就是不好看啊！而且每次开一个新窗口好麻烦。

我又想要一个可可爱爱的便签工具，又想让 CC 能帮我管那些很随意的、草稿本一样的小任务碎碎念——怎么办呢？

那就自己 vibe 一个呗！

## 它能干嘛

CC 作为内置小助理，可以帮你：

- **随手记** — 碎碎念、待办、灵感，打标签归类，置顶、搜索、看板随便切
- **智能查找** — 描述一下你想找啥，CC 帮你从一堆便签里翻出来
- **整理标签** — CC 看看你的便签，告诉你哪些标签该合并、该新增
- **任务图谱** — 便签之间的关联关系，画成图，全屏看贼爽
- **今日摘要 / 本周总结** — 每天晚上自动帮你汇总，周日生成一周深度总结
- **任务盲盒** — 不知道先干啥？选几个标签随机抽一条，让 CC 帮你做决定
- **定时提醒** — 设个时间，到点弹通知提醒你
- **25 款主题** — 焦糖布丁、慵懒晨光、薄荷生巧、莓莓粉荔……挑花眼那种，还能自定义配色

没装 CC 也能用哦，便签管理、标签、搜索、看板、换主题这些基础功能完全不依赖 AI。

## 重点来了：习惯卡池！！

这个是我最最最喜欢的功能。

玩法是这样的：**上传任何一本书的 ZIP 文件，CC 会读完整本书，从里面提炼出 30 条可以每天做的习惯，生成一个专属卡池。** 然后每天随机抽一张习惯卡，做完打卡，慢慢把打卡日历填满～

但它不只是简单的"每天做 XX"打卡。

**你上传什么书，就能变成什么样的人。**

我试了好多本：

**《如何快乐生活》** → 生成了一堆日常快乐小技巧：「给一个陌生人微笑」「写下三件感恩的事」「放下手机走出门」……温温柔柔的。

**《碳水化合物健康食谱》** → 直接变成饮食习惯打卡：「先吃蛋白质再吃主食」「用粗粮替代一次精米」「记录今天的饮水量」……吃货养生版。

**《花卉培养手册》** → 我瞬间变成了温柔的植物系少女：「每天认识一种新植物」「给你的花儿晒晒太阳」「观察叶片的变化并拍照」……

**《占星入门指南》** → 画风突变！变成了自我觉察练习：「检视反复出现的行为模式」「列出消耗你能量的事」「记录让你感到安全的瞬间」……

**《正念禅修》** → 安静的力量：「晨起静坐五分钟」「吃饭时只吃饭」「睡前放下所有念头」……整个人都 zen 了。

然后我手贱传了一本 **《武则天正传》**——

结果它竟然给我生成了一套 **权谋心法** 习惯卡池：「观察身边人的真实动机」「记录一次隐忍不发的瞬间」「复盘一个决策的得失」……每天打卡学着像武则天一样活着，我真的会谢。

还有更多的书——诗词集、《君主论》、甚至是《诡计指南》……又会生成什么样的习惯呢？

**去 [PDF2Skills](https://github.com/AnselKocen/PDF2Skills) 下载书籍 ZIP 吧！** 生成属于你自己的习惯卡池，坚持打卡，把日历贴满～

## 怎么用

### 先准备好这两样

1. **Claude Code CLI** — 你得在本地至少登录过一次。为了保险，打开终端跑一下：
   ```bash
   claude login
   ```
   需要 Anthropic Max 订阅或 API Key。（不装也行，便签基础功能照样能用，就是 AI 功能没法用）

2. **Node.js v18+** — 看看装了没：
   ```bash
   node -v
   ```
   没有就去 [nodejs.org](https://nodejs.org) 下一个 LTS 版本。

### 安装

两种方式，挑一个喜欢的：

**方式一：直接下 App（Mac 用户）**

去 GitHub Releases 下最新的 `.dmg`，双击，拖进应用程序文件夹，搞定。

👉 [去 Releases 下载](https://github.com/AnselKocen/pls_collect_my_task/releases)

**方式二：clone 源码跑（Mac + Windows 都行）**

```bash
cd ~/Desktop                # 进你想放的文件夹
git clone https://github.com/AnselKocen/pls_collect_my_task.git
cd pls_collect_my_task
```

Mac / Linux：
```bash
chmod +x start.sh stop.sh
./start.sh
```

Windows：
直接双击 `start.bat`，或者终端里：
```powershell
.\start.bat
```

跑起来之后浏览器会自动弹开 `http://localhost:3013`，就可以玩了！
